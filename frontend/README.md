# SkillForge AI — Frontend

## Overview

The frontend is built with **React (Vite)** and focuses on a clean, professional UI/UX with role-based navigation and consistent pastel-gradient theming.

## Features

* Responsive landing page with scroll navigation
* Role-based login (Learner/Admin/Guardian)
* Learner Dashboard (complete)
* Admin & Guardian dashboard UI skeletons
* Adaptive content selection UI
* Secure API integration

frontend/

## Run Locally

```bash
npm install
npm run dev
```

App runs at: [http://localhost:5173](http://localhost:5173)
